Features

  • Embed Flash animations and videos in your decks
  • Vertically stretchy background for a clean look at ANY screen size
  • Edit your deck in any text or HTML editor
  • Relative measurements (em's and %'s) mean proper display, no matter what the screen size
  • Styled outline and navigation controls
  • Share your deck by uploading the file -- no need to "export"
  • Plays nice with the Firefox AutoHide add-on for full-screen mode

Key Command Reference

  • Go forward: Click mouse or press [Space], [Return], [Enter], [Right arrow], [Down arrow], [PgDown],
  • Go back: press [Left arrow], [Up arrow], [Page up]
  • Jump to first / last slide: press [Home] / [Enter]
  • Jump directly to a slide: Type the slide number, then hit [Return] or [Enter]
  • Skip forward *n* slides: Type number of slides to skip then hit any "go to next" key (except [Return] or [Enter])
  • Skip backward *n* slides: Type the number of slides to skip, then hit any "go to previous" key
  • Switch between slideshow and outline view: press [T], [O], or click the "outline" button
  • Show/hide slide controls: press [C] or move the mouse to the bottom right
